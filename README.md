## 原型设计

[POP](https://popapp.in/)

- POP 是一款操作非常简单，先用手机拍下草图原型，然后链接到哪个页面，添加跳转链接，之后就变成可演示交互原型的APP，POP 并不提供设计原型的任何UI组件。

[墨刀](https://modao.cc/)

- 墨刀是一款在线的原型设计工具。

[Mockplus](https://www.mockplus.cn/)

- Mockplus 也是一款快速原型设计工具。

以上工具来自：http://keeganlee.me/post/practice/20160816
